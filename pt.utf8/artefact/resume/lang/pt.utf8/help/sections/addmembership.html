Associações Profissionais

Esta secção permite-lhe criar um registo das Associações Profissionais a que pertence, por exemplo: Ordem dos Médicos ou Associação de Profissionais de Educação de Infância. Clique em 'Adicionar' para inserir uma entrada na sua experiência profissional. Utilize 'Cancelar' para fechar o formulário sem guardar a entrada. Se tiver mais de uma entrada, utilize as setas para as ordenar como desejar.

Nota: É importante guardar cada nova entrada utilizando o botão que está no fundo do formulário.

As entradas são mostradas em formato tabular. Altere uma entrada clicando em Editar. Quando terminar, salve as suas alterações para regressar para esta página. Clique em Eliminar para remover entradas.

Se forneceu uma descrição da associação, clique sobre o título para ver ou esconder esta informação.
