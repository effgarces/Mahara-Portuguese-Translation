Permitir páginas públicas de instituições

Se desejar permitir que os utilizadores pertencentes a esta instituição possam criar páginas que poderão ser tornadas públicas, esta opção deverá ser selecionada.

Se esta caixa não estiver selecionada, os utilizadores desta instituição ainda poderão criar páginas públicas se pertencerem a uma outra instituição que as permita.

As páginas públicas apenas são editáveis pelo proprietário.
