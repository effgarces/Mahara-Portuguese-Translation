Tags

Pode adicionar Tags nas páginas que criou. As Tags são etiquetas ou palavras-chave que pode usar para classificar e pesquisar por páginas.

Seleccione em Exibir as minhas Tags irá gerar uma lista de tags que utilizou anteriormente e o número de itens que foram aplicados. Ao seleccionarmos uma Tag nesta lista iremos adicioná-la como uma Tag desta página ou então removê-la no caso de já a termos adicionado-a antes.
