Servidor SMTP

Se quiser forçar o Mahara a utilizar um servidor SMTP específico em vez do do sistema, introduza o endereço aqui.

É possível especificar mais do que um anfitrião separando-os com ponto e vírgula, ex. smtp1.example.com;smtp2.example.com, mas não se esqueça que todas as outras definições (ex. credenciais de autenticação e números de porta) irão aplicar-se a todos os servidores listados. Não é possível especificar credenciais diferentes para cada servidor nesta lista. Esta funcionalidade é útil quando não é necessária autenticação no servidor SMTP ou quando possui diferentes endereços para o mesmo servidor de correio eletrónico, altura na qual todas as diferentes definição continuarão a funcionar.

Este campo pode estar desativado se for sobreposto por uma definição no ficheiro config.php.
