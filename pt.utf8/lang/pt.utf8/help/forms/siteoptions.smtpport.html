Porta SMTP

Se o servidor SMTP utiliza uma porta diferente da 25, pode especificá-la aqui.

Quando a encriptação está ativada as portas por omissão são a 465 para SSL e 587 para TLS. Apenas precisa de especificar a porta se for diferente destas. Verifique as definições corretas com o seu fornecedor de correio eletrónico.

Este campo pode estar desativado se for sobreposto por uma definição no ficheiro config.php.
