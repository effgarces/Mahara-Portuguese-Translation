Método de Autenticação

Todos os utilizadores têm um método de autenticação padrão no Mohara. Em alguns casos especiais, os utilizadores podem ter uma segunda opção de autenticação, no entanto a maioria dos casos é apenas feito de uma única forma.

Poderá escolher qual é a forma de autenticação do utilizador no Mahara. Na maioria das vezes não necessita de alterar esta opção.

Em algumas situações, talvez seja necessário que um utilizador seja autenticado no Mahara usando um método diferente daquele usado anteriormente.Por exemplo, se costumavam usar LDAP mas o servidor do LDAP não tem informação sobre o utilizador e se ele deseja continuar a ter acesso à conta, então pode configurar um método diferente como interno (sem instituição). O utilizador irá ter acesso automatico no Mahara sem usar os recursos do servidor LDAP, no entanto deve de usar o campo "Esqueçeu nome de utilizador/senha" para que possam atribuir uma nova senha.
