Editor HTML

Um editor HTMLis está disponível em algumas secções do sítio. Este editor é um editor WYSIWYG: ‘What you see is what you get’ pelo que permite aplicar formatação ao texto, p.ex. negrito ou itálico. Se o editor for desactivado, apenas pdoerá introduzir texto puro (não formatado).

Esta configuração pode ser inibida pelo administrador do sítio.

Nota:Os editores HTML podem ter uma aparência e comportamento diferentes consoante o browser em uso.
