Educação e Formação

Esta secção permite-lhe criar um registo das suas qualificações académicas. Clique sobre 'Adicionar' para incluir uma nova entrada. Utilize 'Cancelar' para encerrar o formulário sem guardar a sua entrada. Se tiver mais do que uma entrada, utilize as setas para ordená-las como desejar.

Nota: É importante guardar cada nova entrada utilizando o botão que está no fundo do formulário.

As entradas são mostradas em formato tabular. Altere uma entrada clicando em Editar. Quando terminar, salve as suas alterações para regressar a esta página. Clique em Eliminar para eliminar entradas.

Se providenciou uma descrição de uma qualificação, clique sobre o nome da qualificação para ver ou esconder esta informação.
