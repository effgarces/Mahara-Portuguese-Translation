Editor HTML

Um editor HTML está disponível para uso em algumas secções do site. Este editor é chamado um editor WYSIWYG ‘What you see is what you get’. Isto permite-lhe aplicar formatação ao seu texto, por exemplo: texto a negrito ou itálico. Se o editor for desligado apenas poderá introduzir texto simples, sem qualquer formatação.

Nota: Os editores HTML podem ter aparência e comportamento diferente consoante o browser (navegador) utilizado.
