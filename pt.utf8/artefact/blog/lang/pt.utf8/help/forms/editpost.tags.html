Tags

Pode adicionar Tags a Artefactos e Páginas que crie no Mahara. As Tags permitem-lhe adicionar rótulos descritivos aos Artefactos e Páginas e criar um índice de classificações de Tags.

Para associar múltiplas Tags a um Artefacto ou Página, separe cada Tag com uma vírgula. Por exemplo - 'Teste, Semestre 2, Gestão'.

Clicando em Exibir as minhas Tags irá gerar uma lista de Tags usadas anteriormente no seu portefólio juntamente com o número de vezes que foram utilizadas. Clicando numa Tags existente irá adicioná-la á lista de Tags do Artefacto ou Página que está a criar ou editar.

Se utilizar Pesquisar o Meu Portefólio poderá encontrar todos os Artefactos e Páginas que associou a uma Tag em particular, criando desta forma um sistema de arquivamento ou índice para o seu portefólio.
