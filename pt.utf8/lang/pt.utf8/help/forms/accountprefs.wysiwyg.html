Editor HTML

Um editor HTML está disponível para uso em algumas secções do site. Este editor é chamado um editor ‘What you see is what you get’ (WYSIWYG). Isto permti-lhe aplicar formatção ao seu texto, por exemplo: texto a negrito ou itálico. Se o editor for desligado apenas poderá introduzir texto simples, sem qualquer formatação.

Nota: Os editores HTML podem ter aparências e funcionamentos diferentes consoante o browser (navegador) utilizado.
